# book-blog-clien
本项目为Book Blog的客户端。使用vue框架构建，监听3000端口，并向8080端口的服务端发送请求。

## Install
```
npm install fentender/book-blog-client
```

## Run
```
cd book-blog-client
```
### 1.安装npm依赖包
```
npm install
```
### 2.编译和热加载客户端
通过`http://localhost:8080`地址访问博客
```
npm run serve
```
### 3.生成博客静态文件
```
npm run build
```
